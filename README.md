# 📖 QuranVerse – Modern Quran Web Application

QuranVerse is a modern **full-stack Quran web application** built with **Next.js (Frontend)** and **Node.js/Express (Backend)**.

It provides a clean, fast, and interactive experience to read the Quran, search verses instantly, and customize reading settings.

## 🛠 Tech Stack

### 🎨 Frontend
- ⚡ Next.js (App Router)
- 🟦 TypeScript
- 🎨 Tailwind CSS
- 🔄 Redux Toolkit (RTK Query)
- 🧠 Context API (Settings)
- 🎛 HeroUI Components

### ⚙️ Backend
- 🟢 Node.js
- 🚀 Express.js
- 🍃 MongoDB (Mongoose)
- 🔗 RESTful API

### ☁️ Deployment
- ▲ Frontend: Vercel
- ▲ Backend: Vercel
- 🗄 Database: MongoDB Atlas

## ✨ Features

### 📖 Quran Features
- 📚 View all 114 Surahs
- 📜 Read full Surah with verses
- 🌐 Translation support
- 🔍 Live Ayah search (instant suggestions)
- 🎯 Search result highlighting
- 📌 Jump to specific Ayah

---

### 🎛 Customization Features
- 🔤 Arabic font selection (Amiri / Noto)
- 🔎 Adjustable Arabic font size
- 🌍 Adjustable translation size
- 💾 Settings persistence (localStorage)

---

### 🚀 Advanced Features
- ⚡ Debounced live search (optimized)
- 🧠 Global + Surah-based search
- 🎯 Highlight selected ayah in results
- 🔄 Pagination for verses
- 📱 Fully responsive design
